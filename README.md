## 🎯 Featured Projects

### Street Keeper — Final Year Project
Full-stack GPS analytics application that connects to Strava, takes a runner's GPS data, and matches it against real street maps to tell them exactly which streets they have completed, partially run, or not yet started. Built two distinct matching engines iteratively, improving from ~85% accuracy in V1 to 90% with zero false positives in V2.

**Tech:** React 19, TypeScript, Node.js, Express 5, PostgreSQL, Prisma, pg-boss, Vitest, Docker, Strava API, Overpass API, Mapbox API

### [Virtual Pet](https://github.com/devevangel/my-virtual-pet)
Interactive web application featuring a custom lightweight JSON database and a JavaScript-based conversational engine that parses user prompts and returns contextual responses using pattern matching.

**Tech:** JavaScript, Custom JSON DB, DOM Manipulation

### [Hue Smash](https://github.com/devevangel/hue-smash)
Open-source collaborative web game designed to give fellow Software Engineering students a hands-on collaborative coding experience. Led the project: defined feature scope, reviewed contributions, resolved merge conflicts, and maintained code quality.

**Tech:** JavaScript, Canvas API, Git workflows

### Kevvlar *(Private Repository)*
All-in-one project management SaaS platform with real-time video conferencing, live chat, task management, and collaborative workspaces. Sole developer, building the entire application from co-founder's Figma designs. Integrated Stripe for subscription billing and Twilio for real-time communication.

**Tech:** React, Redux, Node.js, Express, MongoDB, Redis, WebSockets, Twilio, Stripe, AWS, Firebase

## 🔧 Hardware & Embedded Projects

### 6-Axis Gyro Stabilisation Module
Designed and built a 6-axis IMU-based stabilisation module for real-time attitude correction in model rockets. Computes corrective outputs to counteract roll, pitch, and yaw deviations during flight. Functional prototype, bench tested.

**Tech:** IMU sensors, embedded programming, sensor fusion, PID control, Fusion 360

### Project Sentinel — Autonomous UAV
Designed a fixed-wing UAV for security surveillance applications. Airframe modelling in Fusion 360 and SolidWorks, sensor payload planning, and research into autonomous navigation (GPS waypoint following, return-to-home logic). Design stage.

**Tech:** Fusion 360, SolidWorks, flight controller research
